This project using plugin [flutter_webview_plugin](https://pub.dartlang.org/packages/flutter_webview_plugin).

```
Widget buildWebview(){
    return WebviewScaffold(
        url: "https://www.google.com",
        appBar: new AppBar(
        title: new Text("Widget webview"),
        ),
    )
}
```
